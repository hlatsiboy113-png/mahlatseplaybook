## Prompt Library (Aligned to Learning & Skill Growth)
This library supports my transition from beginner → developer.

### Debugging Prompt (Core Tool)
```
Act as a senior developer.
Explain this error in simple terms, identify the root cause, and provide a
step-by-step fix.
```

### Code Improvement Prompt
```
Improve this code for readability, structure, and best practices.
Explain what you changed and why.
```

### Learning Prompt
```
Teach me this concept like a beginner
explanation
real-world example
coding example
```
